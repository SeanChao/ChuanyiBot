![github actions ci](https://github.com/SeanChao/ChuanyiBot/workflows/Deploy%20Chuanyi%20Emulator/badge.svg)

## Build and Run

You may need to install `git` and `npm` first.

```sh
# clone the repo
git clone git@github.com:SeanChao/ChuanyiBot.git
# install and run
cd ChuanyiBot
npm install
npm start
```
